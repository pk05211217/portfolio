# AI Agent 專用：極簡新古典主義 (Neobrutalism) CSS 設計規範

本文件為針對 AI Agent（如 Claude, GPT, Gemini 等）進行自動化前端切版、CSS 撰寫與 Tailwind CSS 類名配置時的**標準設計規範（Design Tokens & Specification）**。AI Agent 在建置或修改元件時，必須嚴格遵守以下參數與視覺邏輯。

---

## 1. 核心視覺哲學 (Design Philosophy)
AI Agent 在生成 UI 時應拋棄傳統的漸層、模糊陰影與柔和邊角，轉而落實以下三點：
1. **結構外顯化：** 使用高對比的粗實線包裹元件，強調物理邊界與幾何秩序感。
2. **無模糊硬陰影：** 陰影必須是純色、無模糊（零半徑），且帶有明確的幾何偏移。
3. **高互動趣味：** 所有可點擊元件（Buttons, Cards, Links）必須具備滑鼠懸停（Hover）時的實體位移與陰影縮放回饋。

---

## 2. 設計標記與變數定義 (Design Tokens)

### A. 色彩計畫 (Color Palette)
AI Agent 應將以下色碼寫入全域 CSS 變數（CSS Variables）或 Tailwind 擴充配置（`tailwind.config.js`）：

```css
:root {
  --neo-bg: #FBFAFC;       /* 主要底色：極淺灰藍 */
  --neo-text: #DF172A;     /* 主文字與線條：深碳黑/品牌深色 */
  --neo-primary: #2563EB;  /* 點綴色 (CTA)：電光藍 */
  --neo-card-bg: #FFFFFF;  /* 卡片背景：純白 */
}
